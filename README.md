# 🎯 رودمپ تافل ۲۰۲۶ — هدف نمره ۹۵+

یک وب‌اپ فارسی (RTL) برای آماده‌سازی آزمون **تافل iBT نسخه‌ی ۲۰۲۶** با یک برنامه‌ی **۱۲ هفته‌ای** کامل و قابل‌پیگیری. طراحی‌شده برای شروع از سطح **IELTS ~۵.۵/۶** و رسیدن به **TOEFL 95+** (≈ باند ۵.۰ / CEFR C1).

## ✨ امکانات

- **داشبورد**: درصد پیشرفت کل، شمارش معکوس تا روز آزمون، استریک مطالعه، ساعت‌های انجام‌شده، و نمودار پیشرفت هفتگی.
- **رودمپ ۱۲ هفته‌ای**: ۴ فاز، ۶ روز مطالعه در هفته، با چک‌لیست روزانه و **ساعت مشخص برای هر تسک**. پیشرفت در مرورگر ذخیره می‌شود.
- **راهنمای ۴ مهارت** بر اساس فرمت ۲۰۲۶ (Adaptive Reading/Listening، تسک‌های جدید مثل Listen & Repeat، Integrated/Academic Discussion Writing) + جدول نگاشت نمره.
- **کتابخانه‌ی منابع** شامل **Testinno**، ETS رسمی، و منابع رایگان.
- **تایمر تمرین** با زمان دقیق بخش‌های آزمون ۲۰۲۶ (Reading 30′، Listening 29′، Writing 23′، Speaking 8′) و پومودورو.
- **دفترچه‌ی خطا** داخل اپ برای ثبت و مرور اشتباهات هر تمرین (با فیلتر بر اساس مهارت).
- **نوت‌پد کامل** با چند یادداشت، جستجو، سنجاق‌کردن، شمارش کلمه و ذخیره‌ی خودکار — با نمای دو‌ستونه‌ی بهینه برای تبلت/آیپد.
- **قابل نصب روی گوشی (PWA)** و کارکرد آفلاین.
- رابط **فارسی RTL**، حالت روشن/تیره، و طراحی ریسپانسیو (موبایل‌اول).

## 🗓️ ساختار برنامه

| فاز | هفته | تمرکز | ساعت روزانه |
|-----|------|-------|-------------|
| ۱ | ۱–۳ | پایه‌سازی و فعال‌سازی مجدد زبان، واژگان، گرامر | ~۲.۵ ساعت |
| ۲ | ۴–۷ | مهارت‌سازی تخصصی هر بخش (R/L/S/W) | ~۳ ساعت |
| ۳ | ۸–۱۰ | تمرین فشرده و هدف‌گیری نقاط ضعف | ~۳.۵ ساعت |
| ۴ | ۱۱–۱۲ | موک کامل، مرور عمیق، استراتژی روز آزمون | ~۳ ساعت |

مجموع ~۲۰۰+ ساعت مطالعه‌ی هدفمند.

## 🚀 اجرا

```bash
npm install

npm run build    # بیلد تولید

```

## 🛠️ تکنولوژی

React 18 + TypeScript + Vite + Tailwind CSS + React Router + Recharts + lucide-react.

## 📌 نکته

داده‌های برنامه در `src/data/` قرار دارند (`roadmap.ts`, `skills.ts`, `resources.ts`, `examInfo.ts`) و به‌راحتی قابل ویرایش‌اند. پیشرفت کاربر در `localStorage` ذخیره می‌شود.
